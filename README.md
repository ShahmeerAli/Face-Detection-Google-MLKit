A Flutter app implementing real-time face detection using Google ML Kit. The app guides users through randomized facial actions (smile, blink, look left, look right) for identity verification, leveraging the front camera and ML Kitâ€™s face detection API. Features:
Real-time face detection with ML Kit
Challenge-based identity verification (smile, blink, look left/right)
Live feedback on smile, blink, and head pose
Front camera integration with camera package
Custom overlay mask for user guidance

Technologies:
Flutter & Dart
Google ML Kit Face Detection
